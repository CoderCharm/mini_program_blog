# 微信小程序代码

这个博客程序目前只构建了四个页面
```
/page
    -- welcome  欢迎页面
    -- index    首页
    -- cate     文章分类页
    -- detail   详情页
    -- find     查找页面(暂时未实现)
    -- profile  个人中心(暂时未实现)
```

自己按照写Vue后台管理的想法，封装了微信请求方法`wx.request`可参考`/utils/requests.js`
然后在`/api/*`文件中调用各个接口。

分类使用了`swiper`构建一个滑动的页面。

目前整体比较简陋，从通读微信开发者文档到写完页面，没花多长时间，所以很多小程序的一些细节没有用到，也没有注意。
